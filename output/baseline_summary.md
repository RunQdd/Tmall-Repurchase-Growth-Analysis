
## RandomForest Baseline 实验结果

本阶段基于 50,000 条抽样数据完成 RandomForest baseline 建模。
模型在验证集上的 AUC 为 0.6314，说明基础行为特征对复购用户具有一定区分能力。

在默认阈值 0.5 下：
- Precision = 0.1069
- Recall = 0.4274
- F1 = 0.1710

阈值优化结果显示，当前 F1 最优阈值为 0.50，
对应 Precision = 0.1069，
Recall = 0.4274，
F1 = 0.1710。

业务解释：
当前模型能够识别部分真实复购用户，但 Precision 仍偏低，说明误触达比例较高。
因此，当前 baseline 更适合低成本触达场景，例如站内消息、内容推荐和商品提醒；
如果用于优惠券、会员权益等高成本运营动作，需要进一步提升 Precision。
